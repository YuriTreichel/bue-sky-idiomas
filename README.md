# Blue Sky Idiomas - Website Oficial

![Blue Sky Idiomas](https://images.unsplash.com/photo-1522071820081-009f0129c71c?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

Este repositório contém o código-fonte do novo website da **Blue Sky Idiomas**, uma escola de idiomas com mais de 25 anos de tradição em Porto Alegre, especializada em ensino personalizado e focado em resultados.

## 🚀 Sobre o Projeto

O site foi desenvolvido com o objetivo de modernizar a presença digital da escola, oferecendo uma experiência premium, responsiva e dinâmica para novos alunos e parceiros corporativos. O design foca em clareza, profissionalismo e micro-interações que refletem a qualidade do ensino da Blue Sky.

## ✨ Funcionalidades Principais

- **Metodologia Detalhada**: Seção dedicada a explicar o método focado em conversação e turmas reduzidas.
- **Portfólio de Idiomas**: Informações sobre cursos de Inglês, Espanhol, Mandarim, Alemão e Francês.
- **Área Corporativa**: Soluções personalizadas para empresas e treinamento de equipes.
- **Professores**: Apresentação da equipe qualificada.
- **Parcerias**: Rede de convênios e benefícios.
- **Totalmente Responsivo**: Otimizado para dispositivos móveis, tablets e desktops.
- **Performance de Elite**: Desenvolvido com Vite para carregamento instantâneo.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza as tecnologias mais modernas do ecossistema Frontend:

- **React 19**: Biblioteca core para interface.
- **TypeScript**: Tipagem estática para maior segurança e produtividade.
- **Vite**: Build tool extremamente rápida para desenvolvimento.
- **Tailwind CSS 4**: Estilização moderna baseada em utilitários.
- **Motion (Framer Motion)**: Animações fluidas e interações avançadas.
- **Lucide React**: Conjunto de ícones elegantes e leves.
- **React Router DOM**: Navegação SPA (Single Page Application).

## 📦 Como Executar o Projeto

1. **Clonar o repositório:**
   ```bash
   git clone <url-do-repositorio>
   ```

2. **Instalar as dependências:**
   ```bash
   npm install
   ```

3. **Executar em modo de desenvolvimento:**
   ```bash
   npm run dev
   ```
   O site estará disponível em `http://localhost:3000`.

4. **Gerar build de produção:**
   ```bash
   npm run build
   ```
